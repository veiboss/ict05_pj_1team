버튼

네가지 사이즈( .xsmall / .small / .medium / .large)와 스타일을 같이 사용합니다.

버튼 컬러

버튼은 세가지 스타일(.color1 / .color2, / .bdr-color1, / .bdr-color2)과, 앞서 정의된 네가지 사이즈를 조합하여 사용합니다.

태그는 링크이동은 a, 기능버튼은 button을 사용합니다.

버튼의 disabled 상태

TEXT AREA

TEXT AREA의 사이즈정의는 폰트사이즈와 좌우여백이 인풋과 동일 합니다.

기타

on/off radio의 텍스트는 기본 on/off이며 data-on-text/data-off-text를 이용하여 변경할 수 있습니다.
