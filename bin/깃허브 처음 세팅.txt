# 1. 작업할 폴더 생성
폴더 생성 후 우클릭 > Open Git Bash here 클릭 * 안보일 경우 추가 옵션 표시 눌러보기

# 2. 깃허브 주소 복사하기
팀 깃허브 들어가서 code 창에 있는 code 눌러서 깃허브 주소 복사

# 3. 작업 폴더에 깃 다운하기
git clone '복사한 주소 복사하기' * ctrl+v 안되니 우클릭으로 paste로 추가

# 4. 다시 깃배쉬 키기
다운로드가 되었으면 프로젝트 폴더에서 Open Git Bash here 클릭
그러면 옆에 (master)라고 표시 됨

# 5. 로그 확인해 보기
git log 입력시 HEAD -> master, origin/master, 등 표시가 됨
origin 이 붙어있는것은 클라우드 즉 github임

# 6. 브랜치 확인
git branch 입력시 master 브랜치 하나만 표시됨 

# 7. dev 브랜치 동기화
git checkout -b dev origin/dev dev 브랜치 생성 및 이동 후 
origin/dev를 로컬 dev 브랜치에 동기화
